# Electron with vueTS

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Builds and starts the electron app
```
npm run belectron
```

### Starts the electron app without building (faster)
```
npm run electron
```
